# 👗 ReFitly

ReFitly is a personalized fashion recommendation platform that helps users choose outfits based on their body type, style preferences, and occasion.

It aims to solve fashion confusion by providing smart outfit suggestions and styling guidance, especially for users in Tier 2, Tier 3 cities, and small towns.

---

## 🚀 Features

- 🔐 JWT Authentication
- 📧 Email Verification
- 👤 User Profile Personalization
- 👗 Personalized Outfit Recommendations
- 🎯 Body Type Based Suggestions
- 🧠 Style Tips Engine
- 🧥 Digital Wardrobe Management
- ☁️ Cloudinary Image Upload
- 🎛️ Outfit Filters & Categories
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Services
- Cloudinary (Image Upload)
- Nodemailer (Email Verification)

---

## 🎯 Problem ReFitly Solves

Many users struggle with:
- Choosing outfits
- Understanding what suits their body type
- Fashion guidance in small towns

ReFitly helps users by providing personalized outfit recommendations and style guidance.
